Para crear imagen, ejecutar el siguiente build en la carpeta
que tenga el Dockerfile de jenkins blueocean

docker build -t jenkins/blueocean_ci --no-cache . --network host

Para ejecutar y levantar jenkins, lo siguiente. Se especifican
dos puertos, en este caso se expone el 8081 para web
y el 50000 para integraci√≥n.

docker run -p 8081:8080 -p 50000:50000 -v /var/run/docker.sock:/var/run/docker.sock --name jenkinsblue jenkins/blueocean_ci
